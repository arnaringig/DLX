This is an implementation of Donald Knuth´s Dancing Links algorithm 
to solve Sudoku as an Exact Cover problem.

The method is faster than, and superior to the brute force depth first search
algorithm that is somewhat better known.

My main source is Donald Knuth´s paper:
https://www.ocf.berkeley.edu/~jchu/publicportal/sudoku/0011047.pdf
Dancing Links Donald E. Knuth, Stanford University
